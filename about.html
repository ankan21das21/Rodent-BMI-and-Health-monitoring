About the Project

Introduction

The IoT-Based Rodent BMI Estimation and Health Monitoring System is a laboratory-focused project designed to automate the measurement and monitoring of basic health parameters of rodents used in research and academic experiments. The system integrates sensors, a microcontroller, and cloud technology to accurately measure body weight, calculate BMI using the Lee Index, and store data digitally for analysis and visualization. This approach improves accuracy, reduces manual errors, and enables long-term health monitoring.

Problem Statement

In conventional laboratory practices, rodent body weight and length are measured and recorded manually. This process is time-consuming, prone to human error, and inefficient for long-term data tracking and analysis. Manual record-keeping makes it difficult to observe health trends, detect abnormalities early, and manage large datasets. Hence, there is a need for a low-cost, automated, and reliable system to estimate rodent BMI and monitor health efficiently.

Aim and Objectives

Aim: To design and develop an IoT-based system for accurate BMI estimation and health monitoring of rodents.

  • To measure rodent body weight accurately using a load cell and HX711 amplifier
  • To calculate BMI using the standard Lee Index formula
  • To store measured and calculated data in a cloud database (Firebase)
  • To visualize data using a web-based dashboard
  • To reduce manual effort and improve measurement accuracy

Material and Method

Materials Used:

  • Load cell
  • HX711 load cell amplifier
  • Arduino microcontroller
  • Laptop/PC
  • USB communication cable
  • Firebase cloud database
  • HTML, CSS and JavaScript

Methodology:

The load cell measures rodent body weight which is amplified using the HX711 module and processed by the Arduino. The rodent body length is entered manually. BMI is calculated using the Lee Index formula and all data are transmitted to Firebase and displayed on a web dashboard for analysis.

Result and Discussion

The system successfully measured rodent body weight and calculated BMI accurately. Data were stored securely in Firebase and visualized in real time on the dashboard. The system reduced manual errors, improved efficiency, and allowed long-term monitoring and trend analysis.

Use of the System

  • Monitoring obesity and underweight conditions in laboratory rodents
  • Pharmaceutical drug testing by observing BMI changes
  • Toxicity and safety analysis in preclinical studies
  • Metabolic and nutritional research on rodents
  • Long-term health tracking using digital records
  • Supporting ethical animal handling practices
  • Academic and student laboratory research projects

Conclusion

The IoT-Based Rodent BMI Estimation and Health Monitoring System provides a low-cost, reliable, and efficient solution for laboratory animal health monitoring. The integration of IoT, cloud storage, and data visualization makes the system suitable for biomedical research and academic applications.
